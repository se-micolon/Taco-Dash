# Taco Dash – CS50x Week 0

A taco‑collecting game built with Scratch for Harvard's CS50x course.

## 🎮 How to Play
- Use the **arrow keys** (←, ↑, ↓, →) to move the character.
- Collect tacos to increase your score.
- You have **30 seconds** – get as many as you can before the timer runs out.
- Click the start button to begin.

## 🧠 Concepts Used
- Loops, conditionals, variables
- Custom blocks (My Blocks)
- Broadcast messages
- Start screen with background music

## ✅ CS50 Check50 Results
All requirements passed:
- ✅ At least two sprites
- ✅ Uses loops, conditionals, variables
- ✅ Contains a custom block
- ✅ Working timer and game‑over screen

## 📁 How to Run
1. Download the `.sb3` file from this repository.
2. Open [Scratch](https://scratch.mit.edu/projects/editor) and click **File → Load from your computer**.
3. Play!
